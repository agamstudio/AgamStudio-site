# Agam Cybernetic Studio

Welcome to Agam Studio's official repository. This project showcases a futuristic, cybernetic developer portfolio.

## Tech Stack

- **Vite**: High-performance frontend tooling
- **React**: Component-based UI architechture
- **TypeScript**: Type-safe development
- **Tailwind CSS**: Utility-first styling
- **Shadcn UI**: Accessible components

## Getting Started

1. Clone the repository
```bash
git clone <YOUR_REPO_URL>
```

2. Install dependencies
```bash
npm install
```

3. Run development server
```bash
npm run dev
```

## Build

To build for production:

```bash
npm run build
```
